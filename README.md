# 🌍 India Air Quality Monitoring Dashboard (Excel Project)

## 📊 Project Overview

This project showcases an **interactive Excel dashboard** designed to analyze air quality data across various states and cities in India.

The goal is to transform raw CSV/Excel data into **meaningful insights** using data visualization and dashboarding techniques.

---

## 🔍 Key Features

* 📌 State-wise and city-wise pollution analysis
* 📊 KPI metrics: Minimum, Maximum, and Average values
* 🥧 Pollutant distribution (CO, NO₂, PM2.5, Ozone, etc.)
* 🎛️ Interactive slicers for dynamic filtering
* 🗺️ Geographical map visualization of pollution levels

---

## 📈 Insights

* Identified variation in pollution levels across different states
* Compared pollutant concentrations between cities
* Analyzed trends and patterns in air quality indicators

---

## 🛠️ Tools & Technologies

* Microsoft Excel
* Pivot Tables & Pivot Charts
* Data Cleaning & Transformation
* Data Visualization
